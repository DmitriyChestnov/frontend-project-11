## Setup:

1) Clone the repository:

```
git clone git@github.com:DmitriyChestnov/frontend-project-11.git
```

2) Install depencies:

```sh
make install
```

3) Open webpack's develop page:

```sh
make develop
```

4) Build project:

```sh
make build
```
